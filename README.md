# キーボードぼうけん

子供向けキーボード練習ゲーム。ひらがなのローマ字入力を段階的に習得できる。

## 開発

ビルドステップなし。`src/index.html` をブラウザで開くか、ローカルサーバーを立てるだけで動く。

```bash
# 例: Python の簡易サーバー
python3 -m http.server 8000 --directory src
```

## コンテンツの追加・編集

単語・ステージ・アイテムはすべて `src/data/` 配下の JSON で管理している。

| ファイル | 内容 |
|---------|------|
| `src/data/words.json` | ひらがな単語・文章リスト（難易度別） |
| `src/data/stages.json` | ステージ定義 |
| `src/data/worlds.json` | ワールド定義 |
| `src/data/items.json` | 武器・防具・特殊アイテム定義 |

### words.json に単語を追加する

該当する難易度のリストに単語を追記するだけで、ゲーム内でランダム出題される。

追記後は以下のスクリプトで 50 音順に整列しておくと見やすい。

```bash
python3 sort-words.py
```

スクリプトは `src/data/words.json` の全リストをあいうえお順（50 音順）にソートして上書き保存する。ひらがなの Unicode コードポイントは 50 音順に並んでいるため、標準ソートで正しく整列される。
